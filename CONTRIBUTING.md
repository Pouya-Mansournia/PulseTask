# Contributing

Thank you for contributing.

1. Fork the repository.
2. Create a feature branch.
3. Do not commit credentials or personal schedule data.
4. Preserve the Worker/Apps-Script separation of concerns.
5. Test both immediate Telegram acknowledgement and background spreadsheet processing.
6. Update documentation for changed setup or behavior.
7. Open a focused pull request.

Use clear commit messages such as:

```text
feat: add configurable report timezone
fix: prevent duplicate done events
refactor: batch heatmap formatting
Docs: expand Windows setup guide
```
